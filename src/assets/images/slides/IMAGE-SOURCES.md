## Image Processing

All images were processed using the following steps:
1. WebP files converted to JPG format using `sips`
2. Resized to 800px width (maintaining aspect ratio)
3. Renamed to match slide naming convention (`slide-##-name.jpg`)
4. Optimized for web delivery (50-130KB per image)
